Project Approval Process

How the Approval System Works

1. Project Submission

Students upload project details, abstract, and relevant documents.

2. Guide Review

Internal guide reviews the project and provides initial feedback.

3. In-charge Evaluation

Project in-charge evaluates technical aspects and feasibility.

4. HOD Approval

Final review and approval by the Head of Department.

5. Project Approved

Student receives notification of approval and can proceed.

Project Upload Guidelines

Required Documents

  • Project Abstract (PDF format, max 2 pages)
  • Detailed Project Report (DOC/DOCX/PDF format)
  • Source Code (ZIP format, max 50MB)
  • Presentation Slides (PPT/PPTX/PDF format)
  • Screenshots/Diagrams (JPG/PNG format)

Upload Instructions

  1. Login to your student account
  2. Navigate to "Upload Project" section
  3. Fill in all required project details
  4. Upload necessary documents
  5. Submit for approval
